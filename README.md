## Overview
This project implements an intelligent Tic-Tac-Toe game using **Python** and a **rule-based Artificial Intelligence approach (Logic-2)**.  
The AI dynamically evaluates the board and makes decisions using predefined strategic rules instead of relying on stored move tables.

---

## AI Logic Used (Logic-2)
